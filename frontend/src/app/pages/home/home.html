Características

Todo lo que incluye

🔐
JWT Auth

Tokens firmados con HS256. Interceptor HTTP automático y guard funcional en rutas privadas.

🏷️
Categorías

Organiza notas con etiquetas de color personalizado. Badge visible en cada tarjeta de nota.

🔍
Búsqueda avanzada

Filtra por texto, categoría y fechas con debounce de 300 ms. JPA Specifications en backend.

📄
Paginación

Paginación server-side con Spring Data Page<T>. Orden por fecha de edición, creación o título.

Decisiones técnicas

Seguridad

Reactividad

Arquitectura
  • API REST bien estructurada

  • Componentes standalone
